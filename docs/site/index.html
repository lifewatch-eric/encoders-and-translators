01 — Repository layout

Project Structure

Every transformation is fully self-contained in its own folder. The shared/ library holds reusable templates imported by multiple stylesheets.

02 — Supported conversions

Transformations

Each transformation lives in an isolated folder and ships with example input/output pairs and a test suite.

⬆

From EML 2.1.1
To   EML 2.2.0
✅ Stable
Upgrades namespace URIs, normalises pubDate, injects alternateIdentifier, harvests distribution URLs and reshapes project/methods structure.

EML220_to_DataCite401

To   DataCite 4.0.1
🚧 Planned

ISO19139_to_EML220
From ISO 19139
To   EML 2.2.0
🚧 Planned
Converts ISO 19139 geographic metadata (gmd/gco namespaces) into EML 2.2.0, mapping responsible party roles and bounding-box coverage.

03 — Get started

Quick Start

Any XSLT 1.0 processor works. Saxon-HE and Python lxml are the recommended options.

Saxon-HE (Java) 
# Run a transformation
java -jar saxon-he.jar \
  -s:transformations/EML211_to_EML220/examples/input/sample-eml211.xml \
  -xsl:transformations/EML211_to_EML220/xslt/main.xsl \
  -o:output.xml

# Override packageId
java -jar saxon-he.jar \
  -s:your-file.xml -xsl:transformations/EML211_to_EML220/xslt/main.xsl \
  -o:output.xml package-id=my.new.package.id
Python lxml (no Java required) 
# Test against bundled sample
python3 transformations/EML211_to_EML220/test_transformation.py

# Test against your own EML 2.1.1 file
python3 transformations/EML211_to_EML220/test_transformation.py \
  --input /path/to/your-eml211-file.xml \
  --output /path/to/result.xml
CI — validate all transformations 
bash ci/validate.sh                     # run all
bash ci/validate.sh EML211_to_EML220    # run one
04 — Access & governance

Visibility Model

The repository is private by default. Individual stable transformations can be extracted into standalone public repositories using git subtree split.

ScopeDefault visibilityExtraction command
This repository (all transformations) 🔒 Private —
EML211_to_EML220 (stable) 🌐 Extractable git subtree split --prefix=transformations/EML211_to_EML220
EML220_to_DataCite401 (planned) 🔒 Private Available after stabilisation
ISO19139_to_EML220 (planned) 🔒 Private Available after stabilisation
shared/ 🔒 Internal Stays in monorepo always
